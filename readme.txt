# Pet Supplies Store Website

This is a sample website for a pet supplies store. 
It includes information about dog food products, a customer review, and a contact section.
It showcases product images, descriptions, prices. 
The website also features an audio element for a customer review.

## HTML and HTML5 Tags Used

1. <meta>: Used for defining the document's character set and viewport settings.
2. <title>: Sets the title of the webpage displayed in the browser tab.
3. <link>: Includes external CSS stylesheets and the favicon.
4. <header>: Contains the website header with a title and navigation menu.
5. <nav>: Defines a navigation menu.
6. <ul>, <li>, <a>: Create an unordered list for navigation links.
7. <section>: Organizes webpage content into sections.
8. <h1>, <h2>: Define headings.
9. <div>: Creates a division or section in the document.
10. <table>, <tr>, <td>: Constructs a table structure for displaying product information.
11. <img>: Embeds images into the webpage.
12. <button>: Adds interactive buttons.
13. <audio>: Embeds an audio player.
14. <source>: Specifies the audio source for the `<audio>` element.
15. <footer>: Contains the footer with copyright information.
